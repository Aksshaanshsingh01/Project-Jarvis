# Project Jarvis

Project Jarvis is a local AI-powered personal assistant built using Python, Ollama, and open-source Large Language Models.

## Features

- Local LLM integration using Ollama
- Conversational memory
- Custom Jarvis personality
- Python-based architecture
- Version controlled using Git

## Tech Stack

- Python
- Ollama
- Qwen3 8B
- Git
- VS Code

## Future Roadmap

- Persistent memory using SQLite
- Streamlit UI
- PDF Chat (RAG)
- Automation features
- Google integrations
